Halo, saya Andrew

Saya lahir dan dibesarkan di Pekalongan. Saya menyelesaikan pendidikan dasar dan menengah saya di kota kelahiran saya, lalu pindah ke Yogyakarta untuk mengejar pendidikan tinggi saya di Universitas Kristen Duta Wacana.

Namun, takdir membawa saya ke Jakarta, di mana saya saat ini bekerja dan melanjutkan studi saya di BINUS Online University. Perjalanan saya di dunia teknologi dimulai saat saya masih di SMA, di mana saya pertama kali tertarik dengan dunia perkodingan. Ketertarikan ini semakin berkembang saat saya mulai mengenal Python selama kuliah saya.

Sekarang, saya sangat tertarik dengan Web Development dan berdedikasi untuk memperdalam pengetahuan dan keterampilan saya di bidang ini. Saya percaya bahwa dengan kerja keras dan dedikasi, saya dapat mencapai tujuan saya dan memberikan kontribusi yang berarti dalam industri ini.
